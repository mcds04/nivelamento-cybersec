# 🛡️ Nivelamento em Cybersecurity 🐳

Neste projeto, realizei a configuração completa do meu ambiente de desenvolvimento local, estruturado especificamente para estudos e práticas de segurança da informação (Cybersec).

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **WSL2 (Ubuntu)** – Ambiente Linux para ferramentas de segurança.
* **Docker Engine** – Plataforma de conteinerização.
* **VS Code & Git** – Editor e controle de versão.

---

## 🚀 Práticas Realizadas neste Laboratório

1. [x] Instalação do **WSL2 com Ubuntu 24.04 LTS**.
2. [x] Instalação do **Docker Engine** com chaves GPG binárias.
3. [x] Configuração pós-instalação para rodar o Docker sem `sudo`.

---

## 📑 Guia Rápido de Comandos Docker

* `sudo service docker start` — Inicia o motor do Docker.
* `docker run hello-world` — Executa contêiner de teste.
